Read(ref Utf8JsonReader, Type, JsonSerializerOptions)

Reads a JSON value and converts it into a Coordinates object. This method maps the first angle to longitude and the second angle to latitude.

Remarks

The input JSON is expected to be an array with two values: the first being the longitude and the second being the latitude.
